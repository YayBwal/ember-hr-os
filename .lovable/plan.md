## Goal
Delete the Overview page (the `/dashboard` route) and its UI completely without touching any other module's logic or UI. Operations becomes the HR landing page.

## Why this is safe
Nothing else in the app reads dashboard data — `dashboard.tsx` queries its own `["dashboard-kpis"]` key and is not imported anywhere else. Only navigation targets need to be repointed.

## Changes

1. **Delete the route file**
   - Remove `src/routes/_authenticated/dashboard.tsx`.
   - `src/routeTree.gen.ts` is auto-generated by the TanStack Router Vite plugin — it regenerates on next build, so we do not edit it manually.

2. **Remove the sidebar nav entry** — `src/components/app-shell.tsx`
   - Drop the `{ to: "/dashboard", label: "Overview", icon: LayoutDashboard }` item from the nav list.
   - Remove the unused `LayoutDashboard` import if no longer referenced.
   - Repoint the brand `<Link to="/dashboard">` (top-of-sidebar logo link) to `/operations`.
   - In the role-sync `useEffect`: change `navigate({ to: "/dashboard" })` → `navigate({ to: "/operations" })`, and update the HR-from-team-leader branch likewise.
   - In `changeRole`: HR branch navigates to `/operations` instead of `/dashboard`.

3. **Repoint post-auth redirects** — `src/routes/auth.tsx` and `src/routes/reset-password.tsx`
   - Every `navigate({ to: "/dashboard" })` → `/operations`.
   - `emailRedirectTo: \`${window.location.origin}/dashboard\`` → `/operations`.

4. **Voice assistant navigation enum** — `src/lib/voice.functions.ts`
   - Remove `"/dashboard"` from the allowed-paths enum and from the system prompt's page list (no logic change).

## Out of scope (untouched)
- Operations, Financial, Pipeline, Team Leader Hub, Settings, Organization pages — no UI/logic changes.
- KPI computation, payroll, peer review, task flows.
- Any server functions.

## Validation
- `bunx tsgo` passes (no remaining `/dashboard` references).
- Sign-in lands on `/operations`.
- Role switcher (HR ↔ Team Leader) still routes correctly.
- Sidebar shows no Overview item; all other items intact.
